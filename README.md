## ⭐ Projetos Principais

### 🔢 [EngineerCalc](https://github.com/lucasbrandao136/Calculadora_Eng)

> Calculadoras de engenharia elétrica que rodam direto no browser — sem login, sem anúncio, compartilháveis por URL.

**35+ calculadoras** cobrindo os tópicos centrais de circuitos:

- ⚡ CC — Lei de Ohm, divisores, Thévenin, malhas e nós
- 〰️ AC — Impedância, fasores, ressonância, filtros
- 🔋 Potência — Ativa, reativa, aparente, fator de potência
- 🔌 Semicondutores — Polarização de diodos e transistores BJT/FET
- 🎨 Código de cores de resistores
- 🧲 Capacitores e Indutores — série, paralelo, energia armazenada

**Stack:** `Next.js 14` · `React 18` · `TypeScript` · `TailwindCSS` · `Decimal.js`

---

### ✅ [TaskFlow](https://github.com/lucasbrandao136/taskflow)

> Gerenciador de tarefas full-stack com foco em organização real — não é mais um clone de to-do list.

- 🖱️ **Drag-and-drop** entre listas com estado persistido
- 🔐 **Autenticação JWT** com bcrypt — sem bibliotecas de auth externas
- 🎨 **Listas com cores** customizáveis por projeto
- 🖼️ **Upload de avatar** via Cloudinary
- 🏗️ Arquitetura **MVC** no backend, separação clara de responsabilidades

**Frontend:** `Vue 3` · `Pinia` · `Vue Router` · `TypeScript` · `TailwindCSS`  
**Backend:** `Node.js` · `Express` · `TypeScript` · `PostgreSQL` · `JWT + bcrypt`
